📊 Sales Intelligence Hub
📌 Project Description

Sales Intelligence Hub is a data-driven dashboard built using Streamlit. It helps analyze sales performance and provides business insights through KPIs and visualizations.

🚀 Features
🔐 User Login System
📊 KPI Dashboard
📈 Sales Analysis
🗄️ MySQL Database Integration
📉 Data Visualization using Pandas
🛠️ Technologies Used
Python
Streamlit
Pandas
MySQL
📂 Project Structure

sales-intelligence-hub/
│
├── login.py
├── dashboard.py
├── dbconnection.py
├── database.sql
├── requirements.txt
└── README.md

⚙️ Setup Instructions
Step 1: Clone Repository

git clone https://github.com/Lawanya48/sales-intelligence-hub.git

Step 2: Install Dependencies

pip install -r requirements.txt


Step 3: Run Application

streamlit run login.py

📊 Output
Interactive dashboard with KPIs
Sales insights and analysis
